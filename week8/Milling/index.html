CNC Milling

This week, the class went through the basics of milling and how to operate the machinery. So for this assignment
I was required to walk through some of the concepts given in class and use my previous knowledge to add onto the mini
project.

I started my work on Fusion360, once I wanted to design some 3D shape to later do a 2.5D model, using the milling machine.
I first tried a half of a shape that I had already created because I wanted to cast the whole model afterwards. I followed the
steps and recomendations, adding walls to make easier for the casting and pouring of the resin.

However, this model ended up having issues to run in the mods website, powered by MIT, once it would never calculate its
dimensions, therefore not allowing the milling to happen.

_____________________________________________________________________________

I switched it up and tried to mill the model of my final project.

The videos bellow show the process of how to use the MIT mods, all the way to milling the wax block.

Once you get to the screen with all the specifications, select your stl file and proceed to scale it down.
this step scalling is only necessary to match the requirements of your machine, in my case I tried two different sizes.

This next video shows me adjusting the origin for the drill.

Upon following all the previous steps, you should be set to calculate and send the gcode to the printer.

_____________________________________________________________________________

In all my attempts, I had a hard time getting a perfect session, yet this experience made me confident to use
this technique for other applications.
