Project Idea

As someone that grew up with the marvel heroes, I always wanted to have a super power.
Tony Stark was my inspiration this time around, more specifically the technology he produces.

_____________________________________________________________________________

I first started with a self-destructive safe, but upon reflecting on the
project and on my short term memory loss, I noticed that I was creating
nothing more than a bomb (Just like Tony Stark). After this explosive idea,
I moved on to think about a way to manipulate light and the idea of the
hologram came along.

_____________________________________________________________________________

My project is to create a prototype of a hologram, but not the boring pieces
of plastic that sells on Amazon, an actual good one. A hologram that you can
interact with is my idea. I find it particularly interesting because it allows
you to enrich the space around you with thoughts, work, and entertainment.

_____________________________________________________________________________

I plan for the design to be simple because the idea is already pretty defying.
I want to create an open box with mirrors on the top and bottom and a black
metal tube in the center to be the foundation of the projections. In this project,
I will be using baby powder to make the beams visibles and use the black rod to
help manipulate them.

After doing some more research about the topic, I figured that the best option of hologram,
considering time and money, would be the no-logram by Joanie Lemercier (@joanielemercier). This project
consists in a regulation of a microcontroller to control a water pump to create a cloud of mist that will be
then used as a display for a self-made projector.

Project Goals

Overall, the creation of this project is an idea that I have been carrying with me for a long time now.
I find facinating manupulation of light and always wanted to be able to design my version of it. I look
at it as a starting point, yet I see many applications for this experiment, such as better comprehension
of a 3D space and entertainment purposes, which can later result in something bigger.
