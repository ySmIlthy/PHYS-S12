Electronic Output Device

For this assignment, I was required to expand on the previous task and add an output device
for my already running circuit with a sensor. Nevertheless, this page is a documentation of this
process and some other things that were incorporated in this weekâ€™s assignment.

Basically, I decided to add a buzzer to the capacitive sensor. My idea was to make a doorbell,
but I did not have the time to design a case for it to actually be a doorbell. However, my
system and circuit have the same functions as a simple doorbell system.

_____________________________________________________________________________

The buzzing sound in the background is the buzzer beeping as I touch the capacitive sensor.
This can be visualized in the graph as the drop in value caused in the graph is a reflection of
the capacitive sensor feeling the presence of my finger.

_____________________________________________________________________________

In another way to record this decrease in value, an oscilloscope was utilized to measure the
variation in voltage and how that relates to what is going on in the circuit. Essentially, a drop in voltage,
similar to the drop in value, is represented in the oscilloscope every time the sensor is excited.
