Microcontroling Programming

For this assignment I used the sculpture previously created to integrate the properties
of arduino programming. Essentially, this addition of a more complex code to the arduino allows
infinite scenarios, but for my sculpture only a potentiometer setup was necessary.

_____________________________________________________________________________

An arduino board, potentiometer, DC motor, L9110, and a couple wires were utilized to make
this circuit work. The assembling part is the annoying part, considering the small screws in the
L9110 and the ends of the DC motor that may need some soldering to connect the wires,
but the process was straightforward. The image shown below is a sketch of my circuit made on Flastad.

_____________________________________________________________________________

I honestly find arduino programming extremely confusing, but this week’s assignment helped me
understand a bit better the functions and properties of this language. Below you will be able to
see the code that made my potentiometer setup possible. Basically, I followed the tutorial offered on
the class’ website and it worked really well.

                    
                        
                            void setup() {
                            pinMode(3, OUTPUT);
                            pinMode(A0, OUTPUT);  //This will be GND for the potentiometer
                            pinMode(A4, OUTPUT);  //This will be 5V for the pot                       
                            digitalWrite(A0, LOW);
                            digitalWrite(A4, HIGH);
                            }
                        
                        
                            void loop() {
                            int pot_value = analogRead(A2);   //pot wiper is on A2
                            int motor_speed = map(pot_value, 0, 1023, 255, 0); 
                            analogWrite(3, motor_speed);
                            delay(1);
                            }

_____________________________________________________________________________

The first part of the code is the setup of the UNO board, defining the pins and indicating
the ground (GND) and positive voltage to the system (5V).
